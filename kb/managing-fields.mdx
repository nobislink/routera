---

In Routera, you can assign users to the following object types:

- Contacts  
- Companies  
- Deals  
- Tickets  

You can also create custom data fields to store additional information. The available data field types are:

- Single-line text  
- Number  
- Single checkbox (yes/no)  
- Dropdown  
- Date  
- User

### Creating a Data Field

1. Go to **Settings** <Icon icon="gear" /> and open the **Data Fields** tab.  
2. Click **Create Field**.  
3. In the **Object Type** dropdown, select the object you want to create the field for.  
4. Enter a name in the **Field Label** text field. (Optionally, set a specific internal name.)  
5. Select the **Field Type**.  
6. Click **Create Field** to finish.

### Updating a Data Field

1. Go to **Settings** <Icon icon="gear" /> and open the **Data Fields** tab.  
2. Select the object type (e.g., Contacts) from the dropdown.  
3. Find the field you want to update and click its name.  
4. Make your changes, then click **Save**.  

---

### Deleting a Data Field

1. Go to **Settings** <Icon icon="gear" /> and open the **Data Fields** tab.  
2. Select the object type (e.g., Contacts) from the dropdown.  
3. Locate the field you want to delete in the list.  
4. Select the field, click <Icon icon="trash-can" /> **Delete**, and confirm.
